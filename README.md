# Contrive Internal Calculator

Internal calculation tool for prefabricated construction projects.

This calculator is designed for internal use by **Contrive Interior & Construction** to quickly estimate area, material quantities, and cost breakdowns for:
- Aerocon Cabins  
- Puff Cabins  
- Containers  
- Industrial / Commercial Sheds  

## Features
- Dynamic area calculation (structure, walls, roof)
- Internal partition wall handling
- Optional components (tiles, insulation, toilet, kitchen, etc.)
- Manual rate input with automatic total & GST calculation
- Fully responsive (desktop & mobile)
- Light / Dark theme toggle

## Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript
